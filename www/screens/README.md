## Screenshot-Erstellung im Simulator

1. Simulator starten mit iPhone 15 Pro
2. Zur gewünschten Ansicht navigieren
3. `Cmd + S` drücken für Screenshot
4. Bild umbenennen und hier ablegen

## Bildformat

- Format: PNG
- Größe: iPhone 15 Pro Screenshots (393 x 852 px @1x oder höher)
- Die Bilder werden automatisch skaliert
